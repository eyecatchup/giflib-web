# giflib web

Not sure if I'll ever finish this. Right now just a playground for
PureScript and Thermite.

## Setup

```bash
$ bower install
$ npm install
$ halcyon install purescript
```

## Development

```bash
# Running a dev server
$ npm run watch
$ npm run serve
$ open http://localhost:8080

# Repl'ing
$ npm run repl
> :i Main
```
